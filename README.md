Project title:lock in
A web-based platform designed for basketball coaches to manage teams, design plays, and track player performance. 

## 🚀 Features
*   *Playbook Builder*: Interactive court canvas to draw and save tactical plays.
*   *Drill Library*: Categorized training drills for skill development.
*   *Roster Management*: Track player stats, attendance, and positions.
*   *Mobile Responsive*: Optimized to be viewed on smartphones directly from the court.

## 🛠️ Tech Stack
*   *Frontend*: HTML5, CSS3, JavaScript
*   *Deployment*: GitHub Pages

## 📦 Local Installation
To view and run this project on your computer:

1. Clone the repository:
   bash
   git clone https://github.com
   
2. Navigate into the directory:
   bash
   cd lock-in-basketball
   
3. Open index.html in your web browser.
